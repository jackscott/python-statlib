# Proposed Tasks #

## Create a package from several standalone statistical packages ##

Combine the statistical modules `stats.py`, `pstat.py` and `matfunc.py` into a single package called `statlib`. Write a distutil based setup script (`setup.py`) that can be used to install the package in the standard way (`python setup.py install`). Make your setup script able to publish this package via the [Python Package Index](http://cheeseshop.python.org/pypi).

Write a wiki page on this site that lists the functions available in stat.py and matfunc.py.

Resources:
  * [Distributing Python Modules](http://docs.python.org/dist/dist.html) documentation page

  * stats.py, pstat.py, statstest.zip http://www.nmr.mgh.harvard.edu/Neural_Systems_Group/gary/python.html

  * Matfunc http://users.rcn.com/python/download/python.htm

  * See the [Statistical Computing with Python](http://www.astro.cornell.edu/staff/loredo/statpy/) page on other  libraries.
